## **🚀 How It Works**

### **Method 1: Interactive (Easiest)**
```bash
python tools/recipe_input.py interactive
```
- Step-by-step questions
- No files needed
- Perfect for beginners

### **Method 2: Spreadsheet (CSV)**
```bash
python tools/recipe_input.py csv sample_recipes.csv
```
- Edit with Excel or Google Sheets
- Process multiple recipes at once
- Non-technical friendly

### **Method 3: Text File (JSON)**
```bash
python tools/recipe_input.py json sample_recipes.json
```
- Edit with Notepad
- Well-formatted structure
- For more advanced users
